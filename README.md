# Movie AI Curator

AI와 대화하며 나만의 영화 취향을 발견하고 기록하는 영화 큐레이션 서비스.

## 로컬 환경 설정

```bash
pnpm install

# pre-commit hook 활성화 (최초 1회)
git config core.hooksPath .githooks
```
