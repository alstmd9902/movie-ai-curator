# 🎬 Movie AI Curator

AI를 활용하여 사용자에게 영화를 추천해주는 웹 서비스입니다.

## 📌 Project Goal

이 프로젝트는 단순한 영화 검색 서비스를 넘어 AI 기반 추천 경험을 제공하는 것을 목표로 합니다.

또한 다음 기술들을 학습하고 적용하는 것을 목표로 합니다.

- React
- TypeScript
- Vite
- Vercel AI SDK
- React Doctor
- TMDB API

## 🚀 Features

예정 기능

- 영화 검색
- 영화 상세 정보 조회
- AI 기반 영화 추천
- 추천 이유 설명
- 즐겨찾기 기능

## 🛠 Tech Stack
