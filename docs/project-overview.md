# 프로젝트 개요

> 서비스가 무엇을 만들고 왜 필요한지, MVP 범위와 성공 기준을 정리한 제품 방향 문서.

## 한 줄 정의

AI와 대화하며 나만의 영화 취향을 발견하고 기록하는 영화 큐레이션 서비스.

## 왜 만드는가

- 볼 영화를 고르는 데 시간이 너무 오래 걸린다
- 기존 추천 서비스는 이유 설명 없이 영화 목록만 보여준다
- 본 영화, 보고 싶은 영화를 한 곳에서 기록하고 싶다

## 핵심 경험 (이게 없으면 의미 없다)

1. AI가 추천 이유를 이해하기 쉬운 말로 설명한다
2. 사용자가 AI와 대화하면서 자기 취향을 발견한다
3. 검색 → 상세 → AI 채팅이 하나의 흐름으로 이어진다

## 대상 사용자

- 볼 영화를 고르는 데 막히는 사람
- 장르/분위기/감독 기반 추천을 원하는 사람
- 감상 기록을 개인적으로 쌓고 싶은 사람

## MVP 범위 (P0 — 반드시 구현)

| 기능 | 설명 |
|------|------|
| 영화 검색 | TMDB API 기반 검색 + 필터 |
| 영화 상세 | 정보, 출연진, AI 분석 |
| AI 채팅 | 취향 기반 추천 + 이유 설명 |
| 위시리스트 | 보고 싶은 영화 저장 |
| 관람 기록 | 본 영화 + 평점/메모 |
| 즐겨찾기 | 마음에 드는 영화 저장 |

## 추후 고려 (P1 — MVP 이후)

- 취향 분석 리포트
- 친구와 영화 추천 공유
- 영화 클럽/토론

## 성공 기준

- 사용자가 AI 대화만으로 영화를 발견할 수 있다
- 추천 결과에 이유가 함께 표시된다
- 위시리스트/관람기록/즐겨찾기가 유저별로 저장된다
